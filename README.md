# 📚 LitAnalytics - Анализ литературных жанров по годам

Современная интерактивная платформа для анализа литературных жанров, трендов и статистики с премиальным дизайном уровня Apple/Tesla.

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

---

## 🌟 Особенности

### Премиальный дизайн
- ✨ **Минималистичный интерфейс** в стиле Apple/Tesla
- 🎨 **Glassmorphism эффекты** с размытием
- 🌓 **Темная и светлая тема** с плавным переключением
- 📱 **Адаптивный дизайн** (ПК, планшет, телефон)
- 💫 **Плавные анимации** на базе GSAP

### Функциональность

#### 1. **Главная страница (Hero Section)**
- Интерактивный заголовок с анимацией
- Счетчики статистики с эффектом подсчета
- Фоновые анимированные частицы
- Призывы к действию

#### 2. **Анализ жанров**
- Интерактивные графики популярности
- Фильтрация по годам (1500-2025)
- Сравнение нескольких жанров
- Визуализация трендов

#### 3. **Каталог жанров**
- **Эпические жанры**: Роман, Повесть, Рассказ, Эпопея, Притча, Сказка, Очерк, Эссе, Фельетон
- **Лирические жанры**: Лирическое стихотворение, Ода, Элегия, Эпиграмма, Сонет, Послание, Эпитафия
- **Драматические жанры**: Трагедия, Комедия, Драма, Трагикомедия, Мелодрама, Водевиль
- **Лиро-эпические жанры**: Поэма, Баллада, Басня

Каждый жанр содержит:
- Описание и историю
- Известных авторов
- Известные произведения
- График популярности по годам

#### 4. **Аналитика**
- Круговые диаграммы распределения
- Линейные графики трендов
- Анализ роста и падения
- Сравнение по десятилетиям

#### 5. **Таймлайн литературы**
- Интерактивная временная шкала
- 8 литературных эпох (от Античности до Современности)
- 3D визуализация с Three.js
- Ключевые события каждой эпохи

#### 6. **Поиск и фильтрация**
- Умный поиск по жанрам, авторам, книгам, годам
- Быстрая фильтрация по категориям
- Мгновенные результаты

#### 7. **Карта мировой литературы**
- Распределение жанров по регионам
- Известные авторы по странам
- Статистика по регионам

#### 8. **3D Виртуальная библиотека**
- Трехмерная визуализация книг
- Интерактивные полки
- Анимированные книги

#### 9. **Граф связей**
- Визуализация взаимосвязей авторов
- Связи между жанрами
- Интерактивная сеть с D3.js

## 🚀 Быстрый старт

### Требования
- Современный веб-браузер (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- Интернет-соединение (для загрузки CDN библиотек)

### Установка

#### Способ 1: Локальный запуск

1. **Скачайте все файлы проекта**
   
2. **Структура проекта должна выглядеть так:**
```
litanalytics/
├── index.html
├── css/
│   └── style.css
└── js/
    ├── data.js
    ├── theme.js
    ├── animations.js
    ├── charts.js
    ├── genres.js
    ├── timeline.js
    ├── search.js
    ├── map.js
    ├── library3d.js
    ├── network.js
    └── main.js
```

3. **Откройте `index.html` в браузере**
   - Двойной клик на файл `index.html`
   - ИЛИ правая кнопка мыши → Открыть с помощью → Выбрать браузер

#### Способ 2: Локальный сервер (рекомендуется)

Для лучшей работы запустите локальный сервер:

**Python 3:**
```bash
cd litanalytics
python -m http.server 8000
```

**Python 2:**
```bash
cd litanalytics
python -m SimpleHTTPServer 8000
```

**Node.js (http-server):**
```bash
cd litanalytics
npx http-server -p 8000
```

**PHP:**
```bash
cd litanalytics
php -S localhost:8000
```

Затем откройте в браузере: `http://localhost:8000`

#### Способ 3: Live Server (VS Code)

1. Установите расширение "Live Server" в VS Code
2. Откройте папку проекта в VS Code
3. Нажмите правой кнопкой на `index.html` → "Open with Live Server"

## ⚙️ Настройка

### Изменение цветовой схемы

Откройте `css/style.css` и измените CSS-переменные:

```css
:root {
    --accent-primary: #6366f1;      /* Основной цвет */
    --accent-secondary: #8b5cf6;    /* Вторичный цвет */
    /* ... */
}
```

### Добавление нового жанра

Откройте `js/data.js` и добавьте в соответствующий массив:

```javascript
genresData.epic.push({
    id: 'new-genre',
    name: 'Новый жанр',
    category: 'epic',
    description: 'Описание...',
    history: 'История...',
    authors: ['Автор 1', 'Автор 2'],
    works: ['Произведение 1'],
    period: '1800-2025',
    peakYears: [1850, 1900],
    popularity: generatePopularity(1800, 2025, [1850, 1900])
});
```

### Настройка анимаций

В `js/animations.js` измените параметры:

```javascript
gsap.from('.hero-title', {
    duration: 1,        // Длительность
    y: 100,            // Смещение
    opacity: 0,        // Прозрачность
    ease: 'power3.out' // Плавность
});
```

## 🔧 Устранение неполадок

### Проблема: Графики не отображаются
**Решение**: Убедитесь, что есть интернет для загрузки Chart.js с CDN

### Проблема: 3D элементы не работают
**Решение**: Проверьте поддержку WebGL в браузере и обновите драйверы

### Проблема: Анимации тормозят
**Решение**: Закройте лишние вкладки, отключите расширения браузера

### Проблема: Мобильная версия работает неправильно
**Решение**: Используйте современный браузер, очистите кэш
