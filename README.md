# 📚 LitAnalytics - Анализ литературных жанров по годам

Современная интерактивная платформа для анализа литературных жанров, трендов и статистики с премиальным дизайном уровня Apple/Tesla.

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

---

## 🌟 Особенности

### Премиальный дизайн
- ✨ **Минималистичный интерфейс** в стиле Apple/Tesla
- 🎨 **Glassmorphism эффекты** с размытием
- 🌓 **Темная и светлая тема** с плавным переключением
- 📱 **Адаптивный дизайн** (ПК, планшет, телефон)
- 💫 **Плавные анимации** на базе GSAP

### Функциональность

#### 1. **Главная страница (Hero Section)**
- Интерактивный заголовок с анимацией
- Счетчики статистики с эффектом подсчета
- Фоновые анимированные частицы
- Призывы к действию

#### 2. **Анализ жанров**
- Интерактивные графики популярности
- Фильтрация по годам (1500-2025)
- Сравнение нескольких жанров
- Визуализация трендов

#### 3. **Каталог жанров**
- **Эпические жанры**: Роман, Повесть, Рассказ, Эпопея, Притча, Сказка, Очерк, Эссе, Фельетон
- **Лирические жанры**: Лирическое стихотворение, Ода, Элегия, Эпиграмма, Сонет, Послание, Эпитафия
- **Драматические жанры**: Трагедия, Комедия, Драма, Трагикомедия, Мелодрама, Водевиль
- **Лиро-эпические жанры**: Поэма, Баллада, Басня

Каждый жанр содержит:
- Описание и историю
- Известных авторов
- Известные произведения
- График популярности по годам

#### 4. **Аналитика**
- Круговые диаграммы распределения
- Линейные графики трендов
- Анализ роста и падения
- Сравнение по десятилетиям

#### 5. **Таймлайн литературы**
- Интерактивная временная шкала
- 8 литературных эпох (от Античности до Современности)
- 3D визуализация с Three.js
- Ключевые события каждой эпохи

#### 6. **Поиск и фильтрация**
- Умный поиск по жанрам, авторам, книгам, годам
- Быстрая фильтрация по категориям
- Мгновенные результаты

#### 7. **Карта мировой литературы**
- Распределение жанров по регионам
- Известные авторы по странам
- Статистика по регионам

#### 8. **3D Виртуальная библиотека**
- Трехмерная визуализация книг
- Интерактивные полки
- Анимированные книги

#### 9. **Граф связей**
- Визуализация взаимосвязей авторов
- Связи между жанрами
- Интерактивная сеть с D3.js

## 🛠️ Технологии

### Frontend
- **HTML5** - Семантическая разметка
- **CSS3** - Современные стили, Grid, Flexbox
- **JavaScript ES6+** - Модульная архитектура

### Библиотеки (CDN)
- **GSAP 3.12** - Профессиональные анимации
- **Chart.js 4.4** - Интерактивные графики
- **Three.js 0.158** - 3D визуализация
- **D3.js 7.8** - Графы и визуализация данных
- **Font Awesome 6.4** - Иконки
- **Google Fonts** - Шрифты Inter & Poppins
